My Flix Angular - Overview

My Flix Angular App is a full-featured movie database that empowers users to discover, search and curate favorite films. Powered by an Angular frontend, Node.js/Express backend, and MongoDB, it delivers JWT-secured authentication, RESTful API integration, and a fully responsive design for rich, seamless browsing on any device.

Technologies Used

  • Angular
  • Node.js
  • Express
  • MongoDB
  • CSS3

Features

  • JWT-secured user authentication and authorization
  • Angular-powered movie browsing, searching, and detailed views
  • Responsive design for seamless UX on any device
  • RESTful API integration with Node.js/Express and MongoDB
  • User-curated favorites and personalized watchlists

User Problem

Users often struggle to find a comprehensive, user-friendly movie database that allows them to easily search for films, view detailed information, and manage their favorite movies in one place. My Flix Angular addresses this need by providing a robust platform for movie discovery and curation.

Purpose

The purpose of My Flix Angular is to create a dynamic, engaging movie database that enhances the user experience through intuitive design and powerful functionality. By leveraging modern web technologies, it aims to provide a seamless platform for movie enthusiasts to explore, discover, and manage their favorite films.

Accomplishments

Successfully developed a fully functional movie database using Angular, Node.js, Express, and MongoDB. Implemented JWT-based user authentication and authorization, enabling secure access to user profiles and personalized features. Created a responsive design that ensures optimal viewing across devices, enhancing user engagement and satisfaction.
